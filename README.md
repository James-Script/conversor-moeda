# conversor-moeda

# Conversor de Moeda 💱

Este é um projeto completo de conversor de moedas com **backend em Java Spring Boot** e **frontend em Ionic Angular**, que permite converter valores entre diferentes moedas em tempo real.

# 🔧 Pré-requisitos

- Java 17+

-Maven

- Node.js v22.15.0

- Ionic CLI 4.12.0+

- MySQL ou H2 Database
